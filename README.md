How to use : 

ReactCesium - server : 
          1. npm i 
          2. node server.mjs

ReactCesium - client : 
          1. npm i
          2. npm start

ReactCesium - iot-wrapper : 
          1. python3 iot_device.py

NB: requires InfluxDB-service, mosquitto-service and mongodb-service installed 
and properly configured (also telegraf as plus).
